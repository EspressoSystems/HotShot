Expand description

Overarching network behaviour performing:

  • network topology discovoery
  • direct messaging
  • p2p broadcast
  • connection management

Fields§

§gossipsub: GossipBehaviour

purpose: broadcasting messages to many peers NOTE gossipsub works ONLY for sharing messsages right now in the future it may be able to do peer discovery and routing https://github.com/libp2p/rust-libp2p/issues/2398

§dht: DHTBehaviour

purpose: peer routing purpose: storing pub key <-> peer id bijection

§identify: IdentifyBehaviour

purpose: identifying the addresses from an outside POV

§request_response: DMBehaviour

purpose: directly messaging peer
