pub struct PhaseLockHandle<I: NodeImplementation<N> + Send + Sync + 'static, const N: usize> { /* fields omitted */ }
Expand description

Event streaming handle for a PhaseLock instance running in the background

This type provides the means to message and interact with a background PhaseLock instance, allowing the ability to receive Events from it, send transactions to it, and interact with the underlying storage.
